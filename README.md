# TalentAI — Recruitment Marketing Ecosystem

GenAI-powered recruitment marketing platform that creates hyper-personalized candidate experiences.

## Architecture

```
TalentAI/
├── frontend/          Next.js 15 (App Router) + TailwindCSS + TypeScript
│   └── src/
│       ├── app/
│       │   ├── page.tsx                              Home / dashboard
│       │   └── jobs/[jobId]/[candidateId]/page.tsx   Personalized landing page
│       ├── lib/api.ts                                Backend API client
│       └── types/index.ts                            Shared type definitions
│
├── backend/           Python FastAPI
│   └── app/
│       ├── main.py                  App entrypoint + CORS
│       ├── config.py                Pydantic settings (env-driven)
│       ├── routes/outreach.py       POST /generate-outreach
│       ├── services/outreach_agent.py   Claude-powered outreach generator
│       ├── models/schemas.py        Pydantic request/response models
│       └── db/models.py             SQLAlchemy ORM (Candidate, Job, EngagementEvent)
```

## Core Modules

### 1. Hyper-Personalized Landing Pages (Frontend)
Dynamic route `/jobs/[jobId]/[candidateId]` renders a landing page with AI-generated copy tailored to each candidate's profile — skills, experience, and inferred interests.

### 2. Candidate Outreach Agent (Backend)
`POST /generate-outreach` accepts a candidate profile + job description and returns a 3-step personalized outreach sequence (LinkedIn + email) generated by Claude.

### 3. Intent Engine & Analytics (Database)
SQLAlchemy models track candidates, jobs, and engagement events with scoring. Embedding columns are ready for vector search (pgvector / Pinecone integration).

## Quick Start

### Backend
```bash
cd backend
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env        # Add your ANTHROPIC_API_KEY
uvicorn app.main:app --reload
```
API available at `http://localhost:8000` — docs at `/docs`.

### Frontend
```bash
cd frontend
npm install
cp .env.example .env.local
npm run dev
```
App available at `http://localhost:3000`.
